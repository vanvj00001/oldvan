无我

念念佛知见

被批评时，“我”是谁？——念念佛知见的具体操作

关键词：佛知见、无我、破我执、念念觉照、世间修行、六祖坛经、转识成智、妙观察智、平等性智、五蕴非我、心的主人

前言

我们常说“念念佛知见”，但一遇到别人的批评、指责、甚至只是脸色不好，那个“我”就立刻跳出来——委屈、愤怒、辩解、反击。这时候佛知见去哪了？不是不知道，是用不上。

问题出在哪里？出在我们对“无我”的理解太笼统。你问“被批评的‘我’是谁？”——是肉身在被骂吗？是精神在被骂吗？还是有一个叫做“灵魂”的东西在被侮辱？

如果不能把这个“我”拆开看清楚，那么“无我”就只是一句口头禅。这篇文章，我们就借着“被批评”这个最日常的场景，一层一层剥开“我”的洋葱，看看里面到底有什么。然后你会发现：原来那个受伤的“我”，从来就不是一个真实的存在。

当责任心超越能力

当责任心超越能力：从大乘智慧看方志敏的生死抉择

关键词：方志敏、责任心、大乘佛法、菩提心、无我、愿力、怀玉山、清贫、悲智双运、生死抉择

前言

在现代社会的快节奏中，“责任心”三个字常常压得人喘不过气来。职场的KPI、家庭的期待、社会的评价……我们被告知要“负责”，却很少有人告诉我们：当责任心超出了能力边界，当肩上扛着的重量已经压弯了脊梁，我们该如何自处？世人因责任心而苦，苦在“必须扛”与“扛不动”之间的撕扯。

而在中华历史的长河中，总有一些人的抉择让我们震撼。方志敏——这位闽浙赣革命根据地的创始人，在1935年1月的怀玉山突围战中，本已突出重围，却为了接应后续部队，毅然返回敌人的包围圈，最终因寡不敌众被捕，六个月后英勇就义。

从世俗的功利角度看，这是“责任心超出了能力”的极致悲剧。但从大乘佛法的智慧视角看，这恰恰揭示了责任心在超越“我执”之后，所能达到的不可思议的境界。今天，我们试图借着方志敏的最后一刻，来探讨一个每个人都将面对的问题：当责任超出了能力，我们该如何抉择？

前言

菩萨道

菩萨道：应缘而助的医者之心

关键词：菩萨道、应缘而助、医者之心、不攀缘、无私奉献、三轮体空、大乘佛法、利他、智慧慈悲、无我、因果、缘起

前言

前言

初入佛门，尤其是接触大乘佛法，“菩萨道”这个概念往往最先跃入眼帘，也最容易引发诸多朴素而深刻的疑问。世人常诵“菩萨保佑”，将菩萨视为无所不能、有求必应的神明，寄托着消灾解难、富贵平安的深切期望。因此，最直观的理解便是：行菩萨道，即是效仿佛菩萨，去帮助他人、救助众生。这种理解，虽然停留在“相”的层面，即对现象和行为的认知，却并非谬误。它触及了菩萨道最表层的核心——利他。菩萨是一位“大医者”，其医治的对象，不仅限于身体的病痛，更深入心灵的疾苦与外在环境的困厄。这“医者”的比喻，精准地勾勒了菩萨道的本质：以智慧为明灯，以慈悲为药方，疗愈世间种种苦痛。
