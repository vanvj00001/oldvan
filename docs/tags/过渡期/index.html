修行路上的失重感

修行路上的失重感：你不是一个人

关键词：失重感、修行、觉醒、过渡期、破执、焦虑、信心、着力点、正念、自在

前言

如果你正在读这篇文章，也许你正处于某个特殊的阶段——你已经开始觉察，知道了人生不只是追求财色名利；你开始想要改变，想要走一条不一样的路。但奇怪的是，你发现：旧的执念在消散，但新的力量还没长出来；以前相信的那些“成功法则”，现在不那么管用了；心中有了正念，但正念还很脆弱；有一种轻飘飘的、“失重”的感觉。请放心，你不是一个人。这种状态，是几乎每一个真正走在修行路上的人，都会经历的阶段。
